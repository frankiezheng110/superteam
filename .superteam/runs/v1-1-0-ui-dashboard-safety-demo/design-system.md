# Design System Input

Task: add an admin usage dashboard with a guarded cache-reset action.

## Product Context

- Audience: internal operators and support leads
- Surface: admin dashboard
- Priority: clarity, trust, and safe action affordance over marketing polish

## Visual Direction

- Aesthetic: restrained operational UI
- Typography: high-legibility sans for UI and tabular data support for metrics
- Color: cool neutral surfaces with one controlled amber danger accent for destructive actions
- Layout: dense but readable two-column admin layout with strong panel hierarchy
- Motion: minimal-functional only

## Risk Notes

- destructive actions must look intentionally heavier than read-only controls
- metrics and action panels should not visually compete
- success and warning states need clear semantic contrast

## Open Questions

- whether the cache reset should stay on the main dashboard or move into a protected maintenance panel
